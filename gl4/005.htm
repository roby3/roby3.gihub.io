4.1.4. Вибір типу кабелю для підсистеми кампуса

Як і для вертикальних підсистем, оптичноволоконний кабель є найкращим вибором для підсистем декількох будинків, розташованих у радіусі декількох кілометрів. Для цих підсистем також підходить товстий коаксіальний кабель.

При виборі кабелю для кампуса потрібно враховувати вплив середовища на кабель поза приміщенням. Для запобігання поразки блискавкою краще вибрати для зовнішньої проводки неметалічний оптичноволоконний кабель. З багатьох причин зовнішній кабель виробляється в поліетиленовій захисній оболонці високої щільності. При підземній прокладці кабель повинен мати спеціальну вологозахищену оболонку (від дощу й підземної вологи), а також металевий захисний шар від гризунів і вандалів. Вологозахищений кабель має прошарок з інертного газу між діелектриком, екраном і зовнішньою оболонкою.

Кабель для зовнішньої прокладки не підходить для прокладки усередині будинків, тому що він виділяє при згорянні велику кількість диму.
